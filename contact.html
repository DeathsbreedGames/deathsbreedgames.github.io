Contact

If you would like to contribute to any of our projects you can request a pull-merge for the project. You can do this from our Github Page. If you encounter a bug or some sort of issue with any of the projects, please use the bug tracker for the corresponding project that can also be found via our Github Page.

If you wish to contact us for legal issues or anything that was not mentioned, you can send an e-mail to nicolas.ortega.froysa@gmail.com. And I will try to get back to you as fast as possible (make sure to have the project in question as the subject of the e-mail, or it might be set aside as spam).
